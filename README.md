# Readable App Project
This is Udacity React project by Jan Vigar.

## Installation

### Readable API Server
```
cd server
npm install //install packages
node server //launch server
```
Unless modified in `config.js` server will use port 5001

### Readable Client
```
cd Client
npm install //install packages
npm start //launch readable app
```
